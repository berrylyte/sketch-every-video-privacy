Sketch Every Video! collects and stores the following information:

  • Personally Identifiable Information (PII):

    • User account information
    • Usage patterns
  • Location Data:
    • Geographic location
    • IP address
    • Region information
  • User Activity:
    • Videos watched
    • Sketching duration
    • Interaction patterns

    • Website navigation data
